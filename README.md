* 列表 (数组)
  - list=["1","2"]

  - 列表的遍历

  - _列表常用方法_
    - max(), min(), len(), del() ```内置函数```
    - 新增 
      - append(val)
      - insert(index, val)
      - extend(val) ：将可迭代对象[str range() list]中的元素取出然后追加到列表尾部

      - remove(val): 删除第一个值为 val的元素
      - pop(index) : 删除指定下标的元素 
      - del list[index] :删除指定下标的元素 
      - clear : 清空
    - 其他
      - [].index(val) : 查找val第一次出现的下标
      - [].count(val) : 统计val在列表中出现的次数

  ``tip: 列表的这些方法都只是进行浅层操作，不会进入到嵌套元素中``
       
* 元组tuple: 初始化后不能修改，也没有数组的常用方法
  - ('a',1)
  - 索引
  - 切片
* 字典dict --- 键值对
  - 定义 ``key必须是不可变的，val可以是任意类型``
    - {'a':1,'b':'ssa'}
    - dict([('a',1),('b','ssa')])
  - 字典的嵌套
  - 基本操作
    - 增/改 
      - dict[key] = val
      - update({key: val}) 批量修改，也可以增
    - 删 
      - dict.pop(key, no_info) 会返回删除的key对应的val, 如果没有找到，就会返回设置的no_info, 默认是None
      - del dict[key]
      - dic t.clear() 清空
    - 查
      - dict[key] 如果没有对应的val就会报错
      - dict.get(key, no_info) 如果没有找到，就会返回设置的no_info
  - 常用方法
    - dict.keys() 返回所有的key，类型：``dict_keys([key1, key2])``, 这种类型不支持使用下标
    - dict.values() 返回所有的value，类型：``dict_values([key1, key2])``, 这种类型也不支持使用下标
      - dict.items() 返回包含所有的键值对的列表，每个键值对通过元组显示 [(key, value)]
* 集合set ``[可变集合和不可变集合  无序]``
  - 定义: ``{} 这种定义的是空字典类型而不是空set类型``
    - 可变集合``{1, 2, 3} / set()``
    - 不可变集合 ``frozenset(可迭代对象 str、()、{})``
  - 嵌套
    - 集合中只能嵌套不可变的东西``不可变集合、元组等``
      - 原理：集合虽然不能通过下标查询，但底层仍然对每个元素进行了编号，
        这个编号是通过hash函数来确定的，但是这个hash值是通过内容确定，
        也就是说集合中每个元素的内容必须是确定的
  - tip：
    - 集合时无序的，不能通过下标查询, 也不能进行切片操作
    - 自动去重
    - 1和True也会当作同一个，会把其中一个去重, 哪个在前就留哪个
  - 基本操作
    - 增
      - add(val) 将val作为一个元素添加到set中
      - update(可迭代对象) 将可迭代对象中的元素拆开然后一个个加入
    - 删
      - remove(val) 与add对于，如果val不存在就会报错
      - discard(val) 与remove相同只是不会报错
      - pop() `随机`删除一个元素，并返回
      - clear() 清空集合
    - 查： 通过成员运算符来查 [in ;not in]
  - 常用方法
    - A.difference(B) 找出A中不同于B的元素, 并作为一个集合返回
    - A.difference_update(B) 从A中删除于B中相同的元素，修改A 不修改B, 返回值None
    - A.union(B) 合并A B 并返回一个新的集合
    - A.issubset(B) 判断A是否为B的子集
    - A.issubset(B) 判断A是否为B的超集【包含B中所有的元素】
    - A.isdisjoint(B) 判断A B是否有交集
  - 集合的数学运算``交集 并集 差集 对称差集``
    - 并集 ``|`` 
    - 交集 ``&``
    - 差集 ``-``: 只属于s1 不属于s2 的元素的集合
    - 对称差集 ``^``: s1和s2中 排除 交集中的元素 的集合
  > 集合的遍历只能用for，由于它不能用下标，所以while是无法遍历的

## 格式化输出

### 占位符的精度控制```%m.n```
  > m 正： 左对齐 负： 右对齐
  1. 字符串```%m.ns```
     ```
      m: 字符串的最小宽度，如果长度不够就补空格，如果m小于字符串长度就失效
      n：最多输出n个字符，n>字符串长度则不生效，如果小于则直接截断
     ```
  2. 浮点是```%m.nf```
    ```
      m:控制整体的宽度，如果位数不够就用空格补齐，如果m小就失效
      n:尾数精度控制，默认为6，如果n>尾数，则用0补齐，如果n<尾数，进行四舍五入
    ```
  3. 整数是```%m.nd```
    ```
      m: 整数的最小宽度，如果长度不够就补空格，如果m小于字符串长度就失效
      n: 精度控制，最少用n位显示整数，位数不够用0来补，否则就失效
    ```

## 函数

### 参数
- 位置参数[默认]： 按形参的位置传递参数，严格按照参数顺序来确定
- 关键字参数: 根据形参的关键字来传递，无顺序之分
  - 二者混用：位置参数必须在关键字参数之前
- 限制传参
  - ``/`` 前只能用位置参数
  - ``*`` 后只能用关键字参数
  - ``/``必须在 ``*`` 之前
- 参数默认值```必须作为末尾参数，放在所有必写参数的后面``

- 可变参数
  - 可变位置参数
    - ``*args``: args是元组类型
  - 可变关键字参数
    - ``**kwargs``: kwargs是字典类型
    - kwargs默认 {'args':'val'}
  - 可变参数可以和其他参数混用

### 函数的返回值
- ``return value``

### 函数作用域

- 全局作用域
- 局部作用域
  - 局部作用域和局部变量会在函数调用是创建，函数执行结束时销毁

### 函数的嵌套调用
  1. 函数的定义不会执行函数体中的代码，只有在函数执行时才会执行
    所以在testFuncNested()函数执行时才会去看内部的代码，此时所有的函数都以及定义了，所以可以正常执行
  2. 结果
     ```
     ------进入test1------
     ------进入test2------
     ------进入test3------
     ------正在执行test3------
     ------退出test3------
     ------退出test2------
     ------退出test1------ 
     原理就是： 函数调用栈
     ```

- > 函数的递归调用
  - 递归调用首先要确定函数的出口，即递归的终止条件
  - 函数调用自身的同时必须缩小问题规模
  - 递归调用的底层原理是函数调用栈也叫递归工作栈

## 类和面向对象

- 类
  - 命名：大驼峰``Person``
  - ``init(self, val1, val2)`` 初始化方法，创建实例自动调用，添加属性
  - ``Person(val1, val2)`` 创建实例
  - 访问属性
    - ``P1.val`` 访问单一属性， 也可以给实例追加属性【这种实例只能实例来访问】
    - ``P1.__dict__`` 访问所有属性
  - ``type(p1)`` 获取实例对应的类
  - 实例方法
    - ``def speak(self, msg)`` 类中的自定义方法，实例可以直接使用
    - ``p1.speak('ss') / Person.speak(p1, 'ss')`` 实例方法的调用
  - 自定义方法
    - ``def speak(): xxx  p1.speak = speak`` 往实例中添加方法
  - 静态方法
    - 使用``@staticmethod``装饰器装饰的方法
    - 它接受的参数都是自定义参数，不会接收cls这些参数，所以无法访问类和实例的内容
    - 静态方法通常用于定义与类相关的工具方法
    - 这类方法可以通过实例调用：但不推荐，其他语言会报错

  * 继承
    - Person是父类、超类， student是子类
    - 调用方法和属性时在继承链上的体现：子类 ----> 父类
    * 方法重写：在子类中重写父类中的方法
  - 两个常用判断类之间关系的方法
    - isinstance(instance, class) 判断instance对象是否为指定类或其子类[class]的实例
    - issubclass(class1, class2) 判断class1类是否为class2的子类 
  - 多重继承``in lesson11``
    - ``super()``只能调用第一个继承类的方法
    - ``Person.__init()__`` 需要通过这种方法来调用多个父类的方法
  - 三种属性权限
    - ``self.attr`` 公有属性：在类 子类 实例 以及类外部都可以访问
    - ``self._attr``: protected属性： 在类 子类都可以访问, 在外部能访问，但是不推荐
    - ``self.__attr``: 私有属性： 只能在类 内部使用
      - python底层是通过重命名的方式`_person__age`来定义私有属性的
- getter和setter 
  - ``@property`` getter的装饰器
  - ``Attribute.setter`` setter的装饰器

- 魔法方法``在特定场景自动调用的方法 in lesson12``
   >`` __len__
    __str__
    __lt__
    __eq__
    __ne____``
  -  由于实例对象 这种类型没有原始的len方法，或者说实例对象无法使用原始的len方法，会报错，所以需要重写一下
  - 原理： eg： print(p1) 首先会去调用p1.__str__()方法，如果没有就去Person中找，然后去object中找
- object类：所有类都继承了object类
