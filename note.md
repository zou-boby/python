- 提交到签出分支后，将分支合并到main分支
- 在推送到github


## 变量
- 常量 ``大写``
- 标识符 ``数字、字母、下划线``
- 文件命名 ``下划线:lesson_1``
- 变量命名 ``驼峰式:firstName``
- 变量类型判断
  - type(a) ``返回变量a的类型``
  - isinstance(a, str) ``返回true/false``
## 基本数据类型
* 整形
* 浮点型
  - 四舍五入 ``round(num, 2) 2是预留尾数位数``
  - 向上/下取整 ``import math ;  math.ceil() ; math.floor();``
* Bool型 ``True/False``
* 字符串型
  - 创建
    - 单/双引号 ```',"```
    - 空串 ``str()``
    - 多行字符串 ``用''',"""``
  - 切片``索引``
    - arr[1:5:2] 从1号位开始 到5-1=4号位结束，每隔2位截取
    - arr[1:-1] 从后往前，最后一位是的下标位-1
    - arr[::2] 省略写法
  - 拼接
    - "zhan"+"san" 用+进行拼接，但是只能是相同数据类型
    - "张"*10 *可以重复
  - *tip*: 不同类型之间不能``+`` 
  - 类型转换
    - str --> int
      - 只有纯数字的整数字符串才能转成 int类型
    - float --> int ```直接截取整数部分```
    - bool  --> int ``True:1 ; false: 0``
    - float --> bool
    - float --> str
    - bool --> str
    - bool --> float
* 列表 (数组)
  - list=["1","2"]
  - 列表切片 同字符串
  - 列表常用方法
    - 新增 append()
    - 删除 del()、remove()、pop()
* 元组: 初始化后不能修改，也没有数组的常用方法
  - ('a',1)
* 字典 --- 键值对
  - 定义 {'a':1,'b':'ssa'}
  - dict([('a',1),('b','ssa')])


## 运算符与表达式

### 算术运算符
> ```+ - *  /``` 基本运算
> ```// % **```   
> - % 模运算 
> - // 除法取整数
> - ** 幂运算  2**3 2的3次方

### 赋值运算
> ```= += -= *= /= //= %= **=```
> - //=: c//=a  =>   c = c//a
> - **=: c**=a  =>   c = c**a

### 关系运算符
> ```== != >= <=```

### 逻辑运算符
> ```and  or  not```  *与或非*
> - 优先级
>   - not > and > or
 
### 位运算
> - 转化为二进制进行运算 
> - & *按位与*
> - | *按位或*
> - ^ *按位异或*
> - ~ *按位取反*
> - << *左移*
> - \>> *右移*

### 成员运算符
- 判断是否包含在指定序列中
> in  ; not in


### 身份运算符
- 判断两个变量是否引用同一对象
> is ; is not

## 格式化输出
  - print("%d" %7)
  - print("%d %.2f" %(5, 3.444))
