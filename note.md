# 基础语法

## git的使用
- 签出：切换到签出分支
- 提交到签出分支后，将分支合并到main分支
- 在推送到github

## 基本数据类型
- 整形
- 浮点型
- 字符串型
  - 切片
    - arr[1:5:2] 从1号位开始 到5-1=4号位结束，每隔2位截取
    - arr[1:-1] 从后往前，最后一位是的下标位-1
    - arr[::2] 省略写法
  - 拼接
    - "zhan"+"san" 用+进行拼接，但是只能是相同数据类型
    - "张"*10 *可以重复
- 列表 (数组)
  - list=["1","2"]
  - 列表切片 同字符串
  - 列表常用方法
    - 新增 append()
    - 删除 del()、remove()、pop()
- 元组: 初始化后不能修改，也没有数组的常用方法
